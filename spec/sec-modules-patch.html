Source Text Module Records

ExecuteModule ( ) Concrete Method

The ExecuteModule concrete method of a Source Text Module Record _module_ takes no arguments. It performs the following steps when called:

1. Suspend the currently running execution context. 1. Let _moduleContext_ be _module_.[[Context]]. 1. Push _moduleContext_ onto the execution context stack; _moduleContext_ is now the running execution context. 1. Let _result_ be the result of evaluating _module_.[[ECMAScriptCode]]. 1. Let _env_ be _moduleContext_'s LexicalEnvironment. 1. Set _result_ to DisposeEnvironment(_env_, _result_). 1. Suspend _moduleContext_ and remove it from the execution context stack. 1. Resume the context that is now on the top of the execution context stack as the running execution context. 1. Return Completion(_result_).
